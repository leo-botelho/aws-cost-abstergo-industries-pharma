# Projeto de Redução de Custos para empresa farmacêutica com AWS cloud.

Este repositório contém um projeto de arquitetura em nuvem utilizando serviços da Amazon Web Services (AWS) com foco em redução de custos para uma empresa do setor farmacêutico.

## Objetivo
- Reduzir custos operacionais
- Eliminar infraestrutura física
- Aumentar escalabilidade e resiliência

## Serviços AWS Utilizados
- AWS Lambda
- Amazon API Gateway
- Amazon S3 + Glacier
- Amazon SQS
- Amazon SNS

## Estrutura do Repositório
- `report/` – Relatório final do projeto
- `architecture/` – Diagramas de arquitetura
- `cost-estimation/` – Estimativa de custos
- `policies/` – Políticas de configuração
- `docs/` – Referências e documentação
